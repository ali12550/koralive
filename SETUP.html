أداة البث المباشر للمبارياتLive Football Scoring Tool

1 احصل على مفتاح API Get API Key

زيارة RapidAPIVisit RapidAPI
https://rapidapi.com
اختر API-FootballChoose API-Football

ابحث عن "API-Football" واختر الخطة المجانيةSearch for "API-Football" and select the free plan

انسخ مفتاح APICopy Your API Key

انسخ قيمة X-RapidAPI-Key من صفحة المفاتيحCopy the X-RapidAPI-Key value from the keys page

2 أضف المفتاح إلى المشروع Add Key to Project

افتح الملف: app.jsOpen file: app.js

ابحث عن السطر 2 وقم بالتعديل:Find line 2 and edit:

const API_KEY = 'YOUR_API_KEY_HERE';
⚠️ تنبيهWarning

استبدل YOUR_API_KEY_HERE بمفتاحك الفعليReplace YOUR_API_KEY_HERE with your actual key

3 ابدأ الخادم المحلي Start Local Server

Windows:

افتح PowerShell في مجلد المشروع واكتب:Open PowerShell in project folder and type:

python -m http.server 8000

Mac / Linux:

افتح Terminal واكتب:Open Terminal and type:

python3 -m http.server 8000
✅ النجاحSuccess

سترى: "Serving HTTP on 0.0.0.0 port 8000"You should see: "Serving HTTP on 0.0.0.0 port 8000"

4 افتح في المتصفح Open in Browser

http://localhost:8000
✅ تم!Done!

يجب أن ترى الآن التطبيق وهو يعمل بكامل طاقتهYou should now see the app running with live scores

🆘 استكشاف الأخطاءTroubleshooting

المشكلة: "فشل في جلب المباريات"Issue: "Failed to fetch matches"

الحل:Solution:
  • تأكد من اتصالك بالإنترنتCheck your internet connection
  • تحقق من صحة مفتاح APIVerify your API key is correct
  • تأكد من اشتراكك المجاني نشطEnsure your free subscription is active

المشكلة: لا توجد مبارياتIssue: No matches showing

الحل:Solution:
  • جرب تاريخ مختلفTry a different date
  • حدد دوري مختلفSelect a different league
  • أعد تحميل الصفحة (F5)Refresh the page (F5)
