# 🚀 Quick Start Guide - البدء السريع

## English Instructions

### 1. Get Your Free API Key (5 minutes)
- Go to https://rapidapi.com
- Create a free account
- Search for "API-Football" 
- Click "Subscribe to Test" (free tier)
- Copy your API key

### 2. Add API Key to Your Project
- Open the file: `app.js`
- Find this line (line 2): `const API_KEY = 'a6e91f2c5a694d099a0e6293b8b8c9e5';`
- Replace it with your API key: `const API_KEY = 'YOUR_KEY_HERE';`
- Save the file

### 3. Start the Server
**On Windows:**
- Download Python from python.org (or use built-in PowerShell)
- In this folder, right-click and open PowerShell
- Run: `python -m http.server 8000`

**On Mac/Linux:**
- Open Terminal in this folder
- Run: `python3 -m http.server 8000`

### 4. Open in Browser
- Go to: http://localhost:8000
- Done! ✅

## Common Issues

### Issue: "Failed to fetch matches"
**Solution:**
1. Check your internet connection
2. Verify API key is correct
3. Make sure you subscribed to the free tier on RapidAPI

### Issue: No matches showing
**Solution:**
1. Try a different date
2. Select a different league
3. Refresh the page (F5)

### Issue: CORS Error
**Solution:**
- Use the server method (python -m http.server)
- Don't open the file directly

### Issue: Server won't start
**Solution:**
1. Install Python: https://python.org
2. Make sure you're in the correct folder
3. Try port 8080 instead: `python -m http.server 8080`
